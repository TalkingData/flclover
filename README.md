## 目标
* 测试覆盖率100%

## 环境准备
* 操作系统：支持 macOS，Linux，Windows
* 运行环境：建议选择 LTS 版本，最低要求 7.6

## 快速初始化

通过脚手架快速生成项目:

```
$ npm i flclover-init -g
$ flclover-init flclover-example --type=simple
$ cd flclover-example
$ npm i
```

启动项目:

```
$ npm run dev
$ open localhost:7001
```
