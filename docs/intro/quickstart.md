# 快速入门
一步步地搭建出一个 Flclover.js 应用，让你能快速的入门 Flclover.js。

## 环境准备
* 操作系统：支持 macOS，Linux，Windows
* 运行环境：建议选择 LTS 版本，最低要求 7.6

## 快速初始化

通过脚手架快速生成项目:

```
$ npm i flclover-init -g
$ egg-init flclover-example --type=simple
$ cd flclover-example
$ npm i
```

启动项目:

```
$ npm run dev
$ open localhost:7001
```
